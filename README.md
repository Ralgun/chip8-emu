# chip8-emu
A simple chip8 emulator I have created to practice C++ . Needs SDL2 for it to work.<br>
<br>
# Usage
Simply drag and drop a file you want to open or open cmd and write: <br>'**chip8emu.exe path/to/the/game.ch8**'<br>
